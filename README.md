# college Quora
## A project by students of NIT Durgapur

#### This project is made using :

* Frontend :
  * HTML5   
  * CSS5
  * Bootstrap


* Backend :
  * Node JS
  * Express JS
  * MongoDB


* Template Engine :
  * EJS

## To install dependencies use the followimg commands :

> npm install express mongoose ejs body-parser 

## command to run the app

> node app
